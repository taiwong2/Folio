# Folio Hybrid Demo

iOS app that exercises **the actual native MediaPipe EmbeddingGemma runtime** by combining:

- **Folio** via Swift Package Manager (local path)
- **MediaPipeTasksText** via CocoaPods

Source code is shared verbatim with the SPM demo at `../Example/Sources/FolioDemo/`. The
`#if canImport(MediaPipeTasksText)` branches in `DemoState.swift` and `ContentView.swift`
flip on automatically when this target is built — no per-demo duplication.

## Rebuild from scratch

If you ever blow away the generated artifacts (`*.xcodeproj`, `*.xcworkspace`, `Pods/`),
recreate them with:

```bash
brew install xcodegen   # one-time
xcodegen generate
pod install             # CocoaPods must be installed (`brew install cocoapods`)
```

The output is `FolioHybridDemo.xcworkspace` — **always open this, never the `.xcodeproj`**.

## Run

Open `FolioHybridDemo.xcworkspace` in Xcode 26, pick an iOS 26 simulator (e.g. iPhone 16),
hit ▶︎. Or from the command line:

```bash
xcodebuild -workspace FolioHybridDemo.xcworkspace \
           -scheme FolioHybridDemo \
           -destination 'platform=iOS Simulator,name=iPhone 16,OS=26.0' \
           build
```

## Wiring native EmbeddingGemma

1. Download `embeddinggemma-300m.task` from
   [litert-community/embeddinggemma-300m](https://huggingface.co/litert-community/embeddinggemma-300m)
   and put it somewhere the simulator can reach (e.g. drag into the simulator's Files app, or
   add to the app bundle as a resource).
2. Launch the app, pick **EmbeddingGemma native (MediaPipe)** in the Embedder section.
3. Tap **Pick model file…** and select the `.task` file.
4. Pick a document, ask a question. The "Retrieved passages" disclosure should now show
   `cos 0.XX` scores from real on-device 768-dim vectors.

## What this verifies that the SPM demo can't

| Path | SPM demo | Hybrid demo |
|---|---|---|
| OpenAI cloud generator | ✅ | ✅ |
| Apple Foundation Models generator | ✅ (macOS 26) | ✅ (iOS 26) |
| BM25 retrieval | ✅ | ✅ |
| Ollama HTTP embedding (`EmbeddingGemmaEmbedder`) | ✅ | ✅ |
| OpenAI HTTP embedding (`OpenAIStyleEmbedder`) | ✅ | ✅ |
| **Native MediaPipe EmbeddingGemma adapter** | ❌ (`canImport` false) | ✅ |

## Why iOS-only?

`MediaPipeTasksText` ships an iOS XCFramework only — no macOS slice. There's no
workaround without rebuilding MediaPipe from source. Run the macOS-targeted SPM demo
(`swift run FolioDemo` from the repo root) when you want a faster iteration loop and don't
need the native runtime.
